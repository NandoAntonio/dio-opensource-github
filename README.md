# Desafio de Lógica e Colaboração no GitHub — DIO

Este repositório foi desenvolvido para desenvolver o conhecimento prático em versionamento de código e trabalho colaborativo no GitHub proposto pela **Digital Innovation One (DIO)**.

## 🚀 Sobre o Projeto
O projeto possui o propósito: Resolver o desafio técnico coloborar em um projeto no Github e praticar comandos de Git para versionamento do código.

### 📖 Conceitos de Git & GitHub Praticados
Para além do código, a estrutura deste repositório simula um ambiente de desenvolvimento real e colaborativo, onde foram exercitados os seguintes conceitos:

| Recurso | Aplicação Prática no Repositório |
| :--- | :--- |
| **Branches** | Criação de ramificações isoladas para o desenvolvimento de novas *features* |
| **Fork** | Simulação de fluxo de contribuição externa, realizando a cópia do repositório para a conta pessoal |
| **Pull Requests** | Abertura de requisições de alteração para revisar o código antes de integrá-lo ao projeto principal. |
| **Issues** | Rastreamento e documentação de tarefas pendentes ou melhorias a serem implementadas no código. |
| **Projects** | Utilização de painel visual (estilo Kanban) para organizar o fluxo de trabalho |
| **Wikis** | Construção de uma documentação interna complementar para o projeto, detalhando conceitos |
